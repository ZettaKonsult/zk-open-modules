# Authentication

Zetta Konsult's small API towards AWS' (primarily Cognito's) API.

## Responsibilities

### Cognito CRUD
* User pools.
* Client applications.
* User groups.
* Administrator accounts.
* User accounts.

### Authentication

* User registration.
* User authentication.
* CRUD user attributes.
* Password management.
